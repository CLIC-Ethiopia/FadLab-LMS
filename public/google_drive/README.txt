FADLAB GOOGLE DRIVE MOUNT POINT
===============================

This folder is reserved for Google Drive integration. 

Strategies for connection:
1. Local Sync: Use Google Drive for Desktop to sync a specific folder to this location during development.
2. Build Script: Use a script to pull assets from Drive into this folder before deployment.
3. Hybrid: Store "shortcut" files here that redirect to real Google Drive web URLs.

Example URL structure:
"/google_drive/Semester_1/Syllabus.pdf"
