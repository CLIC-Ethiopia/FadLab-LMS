FADLAB FABRICATION REPOSITORY
=============================

Use this folder to store downloadable manufacturing files:
- .STL / .OBJ (3D Printing)
- .DXF / .AI (Laser Cutting)
- .GCODE (CNC Routing)

How to link in the app:
If you place a file named "gear.stl" in this folder, 
the URL in your code/data will be: "/models/gear.stl"
